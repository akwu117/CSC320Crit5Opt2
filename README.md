# CSC320 Critical Thinking Module 5 — Option 2: Get Monthly Temperatures

This repository contains my Java program for CSC320 Critical Thinking Assignment (Module 5, Option 2).  
The program stores and displays monthly average temperatures for a year, then outputs the yearly average, highest, and lowest temperatures.
